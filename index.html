How to create local repository

  1. Right click IT folder using the mouse.
  2. Select "Git Bash Here"
  3. Exacute "mkdir folderName"
  4. Exacute "cd folderName"
  5. Execute "git init"

    how to link local repo with github

  1. Go to new repository page.
  2. fill repository name field and press "Create repository" button.
  3. Exacute "git remote add origin http://github.com / userName/ folderName.git"
  4. Exacute "git branch -M main"
  5. Exacute "git push -u origin main"
